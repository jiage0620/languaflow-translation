## 第2页：项目概述

### 🎯 项目目标
开发一套功能完整的智能翻译网页系统，实现多语言翻译、翻译记忆库、专业语料库和实时字幕翻译功能。

### ✨ 核心价值
1. 提高翻译效率
2. 保证术语一致性
3. 节省API调用成本
4. 支持字幕批量翻译

### 🛠️ 技术栈
- **前端：** HTML5 + Tailwind CSS v3 + 原生 JavaScript
- **存储：** LocalStorage 本地持久化
- **翻译：** 百度翻译 API（JSONP 跨域）
- **部署：** Vercel 一键部署

---

## 第3页：系统架构

### 🏗️ 整体架构
```
用户界面层
    ↓
业务逻辑层（翻译优先级链）
    ↓
数据存储层（LocalStorage）
    ↓
外部API层（百度翻译）
```

### 🔄 翻译优先级逻辑
1. **专业语料库** - 术语优先匹配替换（最高优先级）
2. **翻译记忆库** - 相同内容直接返回缓存
3. **百度API** - 调用第三方翻译服务（兜底）

### 💾 数据存储结构
- 翻译记忆库：原文、源语种、目标语种、译文、时间戳
- 专业语料库：原文术语、译文术语、所属行业分类
- API配置：APP ID、密钥、接口类型

## 第4页：功能一 - 文本翻译

### 📝 功能特性
- 左右双栏布局，支持大段文本输入
- 语种双向互译（中文/英语/日语/韩语）
- 一键交换语言方向
- 翻译结果复制功能

### 🔧 操作流程
1. 选择源语言和目标语言
2. 输入需要翻译的文本
3. 点击"开始翻译"按钮
4. 等待翻译结果（显示来源：记忆库/API）
5. 复制译文或清空重新输入

### 🎨 界面特点
- 响应式布局，自适应各种屏幕
- 加载动画提示
- 翻译来源标识
- 字符统计显示

---

## 第5页：功能二 - 翻译记忆库（核心考核点）

### 🧠 核心功能
- **自动缓存：** 翻译完成后自动保存到本地
- **智能匹配：** 相同原文+语种组合优先读取缓存
- **节省额度：** 减少API调用次数，降低成本
- **永久存储：** 使用LocalStorage，数据持久保存

### 📋 管理功能
- 可视化展示记忆库列表
- 支持搜索历史记录
- 按语种筛选
- 单条删除和一键清空
- 导出JSON文件备份

### 📊 存储结构
```json
{
    "id": "唯一标识",
    "source": "原文",
    "target": "译文",
    "sourceLang": "源语种",
    "targetLang": "目标语种",
    "timestamp": "翻译时间"
}
```

---

## 第6页：功能三 - 专业语料库（核心考核点）

### 📚 核心功能
- **术语管理：** 管理专业领域术语词条
- **强制替换：** 翻译时优先匹配术语，强制替换
- **行业分类：** IT技术、教育培训、医疗健康、通用领域
- **批量导入：** 支持TXT/CSV文件批量导入

### 📝 操作功能
- 手动新增词条
- 编辑已有词条
- 删除词条
- 搜索术语
- 按行业筛选
- 导出JSON文件

### 🔄 术语替换流程
1. 用户输入待翻译文本
2. 系统遍历语料库术语
3. 找到匹配的术语进行替换
4. 替换后的文本进入翻译流程
5. API返回结果后再次进行术语替换

---

## 第7页：功能四 - 实时字幕翻译（核心考核点）

### 🎬 核心功能
- **SRT解析：** 自动识别标准SRT字幕格式
- **时间轴保留：** 只翻译文本，保留时间轴格式
- **批量翻译：** 一键翻译全部字幕内容
- **导出功能：** 导出翻译后的字幕文件

### 📄 SRT格式说明
```
1
00:00:01,000 --> 00:00:04,000
Hello, welcome to our system.

2
00:00:05,000 --> 00:00:08,000
This is a subtitle example.
```

### 💡 使用场景
- 视频字幕翻译
- 直播字幕生成
- 会议记录翻译
- 影视字幕本地化

## 第9页：技术亮点

### ✨ 技术创新点
1. **纯前端架构** - 零服务器成本，易于部署
2. **翻译优先级链** - 语料库 > 记忆库 > API，提高效率
3. **JSONP跨域** - 解决前端调用第三方API的跨域问题
4. **LocalStorage持久化** - 数据本地保存，无需后端

### 📱 用户体验
1. **响应式设计** - 支持手机/平板/电脑
2. **蓝白科技风** - 简约现代的视觉风格
3. **动画过渡效果** - 流畅的页面切换和交互反馈
4. **友好的错误提示** - 所有操作有成功/失败提醒

### 🔒 安全性
1. API密钥本地存储
